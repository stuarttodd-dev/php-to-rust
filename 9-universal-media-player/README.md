# 9 - Universal Media Player

![php-to-rust logo](../images/php-to-rust.png)

This example demonstrates Rust traits and generics by creating a universal media player that can "play" different types of media (audiobooks, video games) through a common interface.

Features:
- `Playable` trait for media types
- Generic `consume_media` function
- Lifetime annotations in `Metadata` struct

## Run

From this folder:

```bash
cargo run
```

Expected output shows playing different media types and metadata description.
