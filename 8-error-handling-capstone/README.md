# 8 - Error Handling Capstone

![php-to-rust logo](../images/php-to-rust.png)

This capstone project demonstrates modular Rust error handling with custom error types, logging, and parsing logic.

Features:
- Custom error handling module
- Logging utilities
- Parser for key-value assignments
- Process module that sums integer values from configs

## Run

From this folder:

```bash
cargo run
```

Expected output shows the sum of parsed integer values from the sample configuration.
