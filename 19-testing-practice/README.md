# 19 - Testing Practice

![php-to-rust logo](../images/php-to-rust.png)

This example demonstrates Rust testing practices, including unit tests for functions, traits, and mocking with trait objects.

Features:
- Clamping function with boundary tests
- Rate limiting trait with mock implementations
- Job function with spy pattern for call logging

## Run Tests

From this folder:

```bash
cargo test
```

Expected output shows passing tests for clamping, rate limiting, and job execution.
